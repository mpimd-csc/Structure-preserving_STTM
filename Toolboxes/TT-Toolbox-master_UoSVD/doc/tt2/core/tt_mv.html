tt_mv

PURPOSE

Matrix-by-vector product in the TT1.0 format

SYNOPSIS

function [res]=tt_mv(mat,vec)

DESCRIPTION 

Matrix-by-vector product in the TT1.0 format
   [RES]=TT_MV(MAT,VEC) matrix MAT by vector VEC product in TT1.0 format
   Please avoid its usage: it will be removed in
