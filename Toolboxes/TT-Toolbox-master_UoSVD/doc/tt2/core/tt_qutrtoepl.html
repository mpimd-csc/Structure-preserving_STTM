tt_qutrtoepl

PURPOSE

returns the multilevel upper-triangular Toeplitz matrix tt generated by the multi-dimensional input vector x

SYNOPSIS

function [tt]=tt_qltrtoepl(x)

DESCRIPTION 

 returns the multilevel upper-triangular Toeplitz matrix tt generated by the multi-dimensional input vector x 
 in the QTT format

 If the size of the input vector is N,
 then the size of the output matrix is N x N.
