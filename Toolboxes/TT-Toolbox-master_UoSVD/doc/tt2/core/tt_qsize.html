tt_qsize

PURPOSE

For a QTT decomposition _tt_,

SYNOPSIS

function [sz]=tt_qsize(tt,s)

DESCRIPTION 

 For a QTT decomposition _tt_,
 k-th core of which is indexed by
 i^{k}_{1} ,..., i^{k}_{s},
 k=1,...,d,
 returns the array of mode lengths
 sz = [n^{1}_{1} ,..., n^{1}_{s};
                 ,...,
       n^{d}_{1} ,..., n^{d}_{s}]
