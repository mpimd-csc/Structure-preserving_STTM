full

PURPOSE

Converts a QTT-Tucker tensor the full tensor

SYNOPSIS

function [a] = full(tt, sizes)

DESCRIPTION 

Converts a QTT-Tucker tensor the full tensor
   [A]=FULL(QTT_TUCKER,SIZES) Converts QTT_Tucker tensor into a tensor
   with mode sizes SIZES. The dimensions should be consistent.
