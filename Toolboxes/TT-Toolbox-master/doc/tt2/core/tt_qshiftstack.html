tt_qshiftstack

PURPOSE

returns a stack S of non-periodic shift matrices

SYNOPSIS

function [tt]=tt_qshiftstack(d)

DESCRIPTION 

 returns a stack S of non-periodic shift matrices
 in the QTT format

 The size of the shift matrices matrices is N x N
 and the output is an N x N x 2N-tensor,
 where N=2^d(1) x ... x 2^d(end)
