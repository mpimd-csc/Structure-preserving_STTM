rank

PURPOSE

Returns TT-ranks of the given tensor in TT-format

SYNOPSIS

function [r]=rank(a,varargin)

DESCRIPTION 

 Returns TT-ranks of the given tensor in TT-format
   [R]=RANK(A) Returns either all ranks of the TT-tensor A

   [R]=RANK(A,IND) Returns the rank with number IND
