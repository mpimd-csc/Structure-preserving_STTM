qtt_mg_interp

PURPOSE

The simplest multigrid operator in QTT as a TT-matrix

SYNOPSIS

function [m]=qtt_mg_interp(d)

DESCRIPTION 

The simplest multigrid operator in QTT as a TT-matrix
   [M]=QTT_MG_INTERP(D) Computes the simplest MG-operator in the
   QTT-format on 2^D grid. Its stencil is
 1,    0
 0.5,    0.5
 0,    1
 0,    0.5
